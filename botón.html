Botó

Un botó és un component que permet a l'usuari executar una acció, com per exemple: enviar un formulari, obrir un diàleg, cancel·lar una operació, etc.

Exemple

Soc un botó dolent

Instruccions

  • Usar l'element <button> per a crear un botó. La regla d'or per a crear components accessibles és usar elements HTML amb la semàntica i la funcionalitat necessària. Cal evitar la creació de components amb ÀRIA tot el que sigui possible.
