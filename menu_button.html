Botò de menú

Un botó de menú és un botó que obre i tanca un menú. El disseny d'aquest botó inclou, amb freqüència, una fletxa o un triangle que apunta cap avall quan el menú està obert i cap a la dreta quan està tancat.

  • Amazon
  • Apple

Instruccions

  1. El botó es crea amb l'element <button>.
  2. El botón necesita el atributo aria-haspopup, que serveix per a informar les tecnologies de suport que el botó obre nou contingut. El valor d'aquest atribut pot ser menu o true.
  3. El botó també necessita l'atribut aria-expanded,que serveix per a informar les tecnologies de suport sobre l'estat del contingut. El valor d'aquest atribut seràtrue si el menú és visible o falsesi el menú està ocult. S'usa JavaScript per a actualitzar el valor cada vegada que l'usuari premi el botó.
  4. El menú es crea amb l'element <ul>.Cada opció del menú serà un element de la llista.
  5. L'element<ul> necessita l'atributrole amb el valor de menu. Aquest atribut serveix perquè les tecnologies de suport identifiquin aquest element com un menú.
  6. El menú està ocult per defecte. Per a això, s'usa CSS per a seleccionar l'element <ul> i declarar la propietat display amb el valor hidden.
  7. S'usa JavaScript per a actualitzar la propietat display de l'element <ul>. Quan el menú sigui sigui visible, tindrà el valor de block. Quan estigui ocult, tindrà el valor dehidden.
