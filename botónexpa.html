Botó expandible

Un botó expandible és un botó que mostra i oculta informació. Aquest component, per tant, consta de dues parts: un botó i el contingut que es mostra i oculta. El botó sol incloure algun símbol (per exemple, fletxes) per a indicar l'estat del contingut (visible o ocult).

Exemple

Instruccions

  1. El botó es crea amb l'element<button>.
  2. El botó necessita l'atributaria-expanded,que serveix per a informar les tecnologies de suport sobre l'estat del contingut. El valor d'aquest atribut seràtrue si el contingut és visible o false si el contingut està ocult. S'usa JavaScript per a actualitzar el valor cada vegada que l'usuari premi el botó.
  3. El contingut es crea amb l'element HTML que sigui més adequat. En els casos més senzills, l'element <div> seria adequat. S'usa JavaScript per a actualitzar la propietat display d'aquest element. Quan sigui visible, tindrà el valor de block. Quan estigui ocult, tindrà el valor dehidden.
