Interruptor

Un interruptor és un component que ofereix a l'usuari dues opcions: actiu i inactiu. Permet a l'usuari percebre de manera ràpida si una opció està activada o desactivada. De la mateixa forma, permet canviar el seu estat ràpidament.

Exemple

Instruccions

  1. L'interruptor es crea amb l'element <button>.
  2. El botó necessita l'atribut role amb el valor de switch per a identificar-ho com un interruptor.
  3. El botó també necessita l'atributaria-checked. El seu valor serà true si l'opció està activada o false si està desactivada. Aquest valor s'ha d'actualitzar, usant JavaScript, cada vegada que l'usuari pressioni el botó.
  4. El botó ha de tenir text descrivint l'opció que es pot activar o desactivar. Per exemple: "manera fosca".
  5. A més del text, el botó necessita contenir alguna manera de transmetre visualment el seu estat (actiu o inactiu). Per exemple, usant SVG per a crear identificadors visuals per a cada estat. Aquest estat s'ha d'actualitzar, usant JavaScript, cada vegada que l'usuari pressioni el botó.
